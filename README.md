# Visualizaing Bardo

This project uses word frequency and co-occurrence network analysis to explore how selected Tibetan Buddhist bardo texts describe the transition from dying to liberation or rebirth.


## Files

- `bardo_project.ipynb`: Jupyter Notebook with code and explanation.
- `bardo_texts_dataset_Jun27.csv`: Dataset of selectied bardo-related texts.
- `README.md`: Project overview and file description.


## Research Question

How do selected Tibetan Buddhist bardo texts deacribe the transition from dying to liberation or rebirth?


## Method

This project uses:
- work frequency analysis
- keyword frequency visualization
- co-occurrence network analysis
- network visualization
